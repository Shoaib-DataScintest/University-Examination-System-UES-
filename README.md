# 🎓 University Examination System

##  Overview

This project is a web-based University Examination System developed using HTML, CSS, Java, and MySQL. It automates the traditional examination process, making it more efficient, secure, and user-friendly.

---

##  Features

* Student registration and login
* Admin panel for exam control
* Online examination system
* Automated result generation
* Database-driven system

---

##  Technologies Used

* HTML (Frontend)
* CSS (Styling)
* Java (Backend logic)
* MySQL (Database)
* phpMyAdmin (Database management)

---

##  Database Setup

1. Open phpMyAdmin
2. Create a new database:

   ```
   exam_portal
   ```
3. Import the provided SQL file:

   ```
   exam_portal.sql
   ```
4. Ensure tables like `users`, `exams`, etc. are created

---

##  How to Run

1. Start your server (e.g., Apache/Tomcat if used)
2. Start MySQL
3. Import the database (see above)
4. Run the Java backend
5. Open the frontend (HTML files) in browser

---

##  Project Structure

* `/frontend` → HTML & CSS
* `/backend` → Java code
* `/database` → SQL file
